(d = sqrt((x2 - x1)^2 + (y2 - y1)^2)) - Находит расстояние между центрами окружностей
(d == r1 + r2) - Расстояние между центрами равно сумме радиусов. Окружности касаются друг друга.
((d == 0) & (r1 == r2)) - Окружности идентичны, имеют равные радиусы и их центры совпадают.
(d + r1 < r2 || d + r2 < r1) - Один из радиусов больше суммы расстояния между центрами и другого радиуса. Одна окружость находится внутри другой.
Если строка 3 и 4 не верны, то окружности пересекаются.
(d > r1 + r2) - Расстояние между центрами больше суммы радиусов. Окружности не взаимодействуют друг с другом.

Тестовые данные
1. Пересечение: x1 = 1 y1 = 1 ; x2 = 2 y2 = 2 ; r1 = 2 r2 = 2 ;
2. Касание: x1 = 2 y1 = 2 ; x2 = 2 y2 = 6; r1 = 3 r2 = 1 ;
3. Одна окружность внутри другой: x1 = 0 y1 = 0 ; x2 = 0 y2 = 0; r1 = 10 r2 = 5 ;
4. Окружности никак не взаимодействуют: x1 = 0 y1 = 0 ; x2 = 3 y2 = 3; r1 = 2 r2 = 2 ;
5. Окружности идентичны: x1 = 0 y1 = 0 ; x2 = 0 y2 = 0; r1 = 2 r2 = 2 ;
